# US001 – Create Field for Campaign/Product Name

## Description
Implement a new editable field in the page configuration to store a campaign/product name.  
This field will be used for user interaction tracking and analytics.

## Acceptance Criteria

### **ACC1 – Backend Field**
- There must be an editable field for the campaign/product name.
- The field must appear under the "Customization" section.

### **ACC2 – Interaction Tracking**
- Track how many users access the campaign page.

### **ACC3 – Association**
- All interactions must be linked to the campaign field value.

### **ACC4 – Data Availability**
- The interaction data must be available for Analytics/Data Lake consumption.
